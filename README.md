# legal

Legal documents for Ysg8App applications. Published via GitHub Pages.

## Structure

```
legal/
├── _config.yml
├── README.md
├── index.md                    # Landing page listing all apps
└── simplehabits/
    ├── index.md                # Simple Habits-specific landing
    ├── privacy-policy-ja.md    # Japanese privacy policy
    └── privacy-policy-en.md    # English privacy policy
```

Each application has its own subdirectory so policies can be added or modified independently as the lineup grows.

## Hosted URLs

- Root index: https://sgy-s-app.github.io/legal/
- Simple Habits landing: https://sgy-s-app.github.io/legal/simplehabits/
- Privacy Policy (JP): https://sgy-s-app.github.io/legal/simplehabits/privacy-policy-ja.html
- Privacy Policy (EN): https://sgy-s-app.github.io/legal/simplehabits/privacy-policy-en.html

## How to Update

1. Edit the relevant `.md` files
2. Commit and push to `main`
3. GitHub Pages will rebuild automatically (usually within a minute)
